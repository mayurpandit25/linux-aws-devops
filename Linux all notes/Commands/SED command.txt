How to show only a given line or range of lines?
sed -n '1p' file_name
sed -n '1,5p' file_name
sed -n '$p' file_name

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
How to see all the users from India Country?
sed -n '/India/p' file_name

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

SED Regular Expressions

^   start of line
$   end of line
.   single character
[]  match character set
[^] exclusive Set 
*   zero or more occurance

examples:

sed -n '/^2/p' file_name
sed -n '/ia$/p' file_name

How to find a 5 letter name which start with S and end with a?
sed -n '/^S...a$/p' names

How to find names start with V?
sed -n '/^V/p' names

How to find names end with a?
sed -n '/a$/p' names

How to use wild card?
ls -ltr *.txt

How to see names start with only A and C?
sed -n '/[AC]/p' names
sed -n '/[AC]/p' names

How to see names start with only A to D?
sed -n '/^[A-D]/p' names

SED command:-

ðŸ”¹ Basic Level

1.Print the whole file (default)
  sed '' sed_sample.txt ------------------>> Does nothing, just prints the file
 
2.Show a specific line (e.g., line 3)
  sed -n '3p' sed_sample.txt

3.Show a range of lines (line 2 to 5)
  sed -n '2,5p' sed_sample.txt


ðŸ”¹ Editing & Replacing

1.Replace first occurrence of a word in each line
  sed 's/Linux/Unix/' sed_sample.txt ---------->> Changes only the first â€œLinuxâ€ in each line to â€œUnixâ€.

2.Replace all occurrences in each line
  sed 's/Linux/Unix/g' sed_sample.txt

3.Case-insensitive replace
  sed 's/linux/Unix/Ig' sed_sample.txt -------->> ðŸ‘‰ I makes it ignore case.

4.Replace only on a specific line (e.g., line 6)
  sed '6s/Linux/Ubuntu/' sed_sample.txt


ðŸ”¹ Deleting

1.Delete a specific line (e.g., line 5)
  sed '5d' sed_sample.txt

2.Delete a range of lines (e.g., line 2 to 4)
  sed '2,4d' sed_sample.txt

3.Delete lines containing a word
  sed '/dog/d' sed_sample.txt


ðŸ”¹ Inserting & Appending

1.Insert a line before line 3
  sed '3i This is a new line before line 3' sed_sample.txt

2.Append a line after line 3
  sed '3a This is a new line after line 3' sed_sample.txt

3.Change entire line (replace full line 5)
  sed '5c This line has been completely replaced' sed_sample.txt


ðŸ”¹ Advanced / Industry Use

1.Replace text only in lines that match a pattern
  sed '/Linux/s/is/was/' sed_sample.txt ------------->> Only changes lines that contain "Linux".

2.Multiple replacements in one command
  sed -e 's/Linux/Unix/g' -e 's/Sed/SED/g' sed_sample.txt

3.Remove extra spaces (convert multiple spaces to one)
  sed 's/  */ /g' sed_sample.txt

4.Show only lines that were changed
  sed -n 's/Linux/Unix/p' sed_sample.txt

5.Save changes to a new file
  sed 's/Linux/Unix/g' sed_sample.txt > newfile.txt

6.Edit file directly (in-place)
  sed -i 's/Linux/Unix/g' sed_sample.txt
